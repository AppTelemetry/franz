Consumer

public class Consumer

A consumer that listens to and consumes messages from certain topics. All consumers belong to a group. Do not initialize a Consumer instance directly, instead use Cluster.getConsumer(topics: groupId:)

See also

    If true the consumer will call the handler for all existing messages, and if false the consumer will only call the handler for new messages.
    handler

    Called whenever a message is received, along with that message.
